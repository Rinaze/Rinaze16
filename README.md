## Диаграмма классов для задания Практика «Поиск в ширину»
![image](https://github.com/user-attachments/assets/ac7ed20e-5a09-4b5a-9d72-50c970e9dab5)

### Описание основных сущностей

1. **BfsTask**
   
*BfsTask* - статический класс, реализующий алгоритм поиска в ширину (BFS) для поиска путей в подземелье.
- FindPaths(Map map, Point start, Chest[] chests) - основной метод, осуществляющий поиск путей от стартовой точки до сундуков. Возвращает перечисление путей, каждый из которых представлен односвязным списком точек.
- GetNeighbourPaths(SinglyLinkedList point, HashSet visit) - вспомогательный приватный метод, генерирующий соседние вершины для текущей точки пути, которые ещё не были посещены.

2. **SinglyLinkedList**
   
*SinglyLinkedList* - обобщённый класс односвязного списка.
- Используется для хранения последовательности точек пути (от начальной до текущей).
- Каждый элемент содержит значение (Value) и ссылку на предыдущий элемент (Previous).

3. **Point**
   
*Point* - класс, представляющий координаты на карте.
- Свойства: X (координата по горизонтали), Y (координата по вертикали).
- Используется для обозначения положения на карте.

4. **Map**
   
*Map* - класс, представляющий карту подземелья.
- Свойство Dungeon - двумерный массив ячеек (MapCell), описывающий структуру подземелья.
- Метод InBounds(Point p) - проверяет, находится ли точка в пределах карты.

5. **Chest**
   
*Chest* - класс, представляющий сундук с сокровищами.
- Свойство Location - точка (Point), где расположен сундук.

6. **MapCell**
   
*MapCell* - перечисление, определяющее тип ячейки карты.
- Возможные значения: Empty (пустая), Wall (стена) и, возможно, другие.
